# Changelog

Alle belangrijke wijzigingen aan dit project worden gedocumenteerd in dit bestand.

## [1.0.0] - 2024-12-19

### Toegevoegd
- Eerste release van CookieConsent & IframeManager integratie
- Ondersteuning voor YouTube en Vimeo video embeds
- Nederlandse en Engelse vertalingen
- Sanity CMS schema integratie
- GDPR compliant cookie toestemming
- Responsive video embeds met verschillende aspect ratios
- Complete installatiehandleiding en documentatie

### Features
- Cookie consent modal met granulaire toestemming per service
- Iframe manager voor lazy loading van video content
- Meertalige ondersteuning (NL/EN)
- Sanity Studio integratie met custom schema
- TypeScript ondersteuning
- CSS custom properties voor styling aanpassingen

### Technische details
- Gebaseerd op vanilla-cookieconsent v3.0.1
- Gebaseerd op @orestbida/iframemanager v1.3.0
- Compatibel met Astro 4.0+
- Node.js 18+ vereist
